Slide&Impress


A WYSIWYG editor for Impress.js


Edit

  • Toggle between presentation and edit mode.
  • Toolbar for editing.
  • Wrap button for easy editing.

Transformations

  • Buttons appended to the slide.
  • 3D transitions added.
  • Input text for setting the transformations.

Others


  • Slideshow for automatic presentation.
  • Debug of builder.js.

Libraries

  • Builder.js(Extended).
  • FileSaver.js.
  • Aloha Editor(Extended).
